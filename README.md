# Blog-website
Blog Website
This is a Simple Website for blogging made using
  1. PHP
  2. Mysql
  3. Javascript
  4. Html
  5. Css
  6. Bootstrap

Features of Website:-
  1. Add Post with image
  2. Search User Based on username with autocomplete facility
  3. View all the post of searched user
  4. Likes the Post
  5. Comment the Post
  6. Notification
  7. Follow a user
  8. There is an admin section in which admin can set permission for user which allow user to post a Blog
  9. Good Login and Signup page with Email Verification
